# 用python编写的小游戏合集
* 目前有五个小游戏和程序了
## PingPingBird
* 用pygame编写的第一个小游戏，以后会不断更新
## Plane
* 打飞机的小游戏，对pygame和面向对象编程有了更深入的了解
## Tetirs
* 俄罗斯方块，学习了[@yeahatgithub][1]的程序，感谢！界面进行了美化
## Maze
* 用回溯和prim算法生成迷宫
## KnightTour
* 用回溯算法解决骑士游历问题
## GoBang
* 用极大极小值和剪枝算法实现五子棋AI
## Sudoku
* 用递归回溯法解决数独，在终盘上随机挖空生成数独

[1]:https://github.com/yeahatgithub
