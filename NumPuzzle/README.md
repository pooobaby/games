# NumPuzzle
数字华容道小游戏

## 游戏说明:
* 用pygame实现数字华容道小游戏
* 游戏的实现还算比较简单，用二维数组存储数字的状态，只要控制好移动时的状态变换就可以了
* 初始状态生成时为了避免死局，采用的随机移动n次，这样能够确保可以正常归位
