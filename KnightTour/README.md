# KnightTour
用pygame编写的练习算法小程序，以后会不断更新

## 游戏说明:
* 问题：用回溯算法解决骑士游历问题，并用窗口直观显示
* 从算法到程序完全是自己完成，对回溯算法有了更深刻的了解

## 系统文件：
- 【back_tracing.py】：主程序，实现回溯算法
- .\images\...：骑士图片，标记图片
- .\fonts\...：字体文件
- .\screenshot\...：程序画面截屏
